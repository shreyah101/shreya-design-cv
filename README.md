## ✦ About The Project

This is a highly artistic, premium web portfolio designed to feel like a **luxury printed magazine**. It departs from traditional portfolio templates, utilizing massive experimental typography, asymmetrical editorial layouts, and a unique rotating vinyl record hero section.

### 🎨 Design Aesthetic
- **Color Palette:** Soft blush pink (`#F5B6D8`), off-white paper (`#F7F5F1`), charcoal ink (`#1A1A1A`), and deep burgundy accents (`#8A1C22`).
- **Typography:** Dramatic hierarchy blending `Cormorant Garamond` (display headings), `Great Vibes` (decorative script), and `Inter` (clean body text).
- **Layouts:** Features alternating full-width project spreads, polaroid-style photography, and subtle CSS noise textures to mimic physical paper.

---

## ✦ Key Features

- **💿 Interactive Vinyl Hero:** A custom CSS-animated rotating vinyl record that creates a striking first impression.
- **🖼️ Magazine Project Spreads:** Each design case study is presented as an expansive editorial spread with massive watermark numbers.
- **🔍 Hardware-Accelerated Slideshows:** Every project opens into a responsive modal containing a custom-built horizontal image slider with thumbnail navigation and full-screen lightbox zoom.
- **📱 Fully Responsive:** The editorial layouts seamlessly adapt from massive 4K monitors down to mobile screens.
- **⚡ Blazing Fast:** Built with React + Vite for instant hot-module-reloading and optimized production builds.

## ✦ Running Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shreyah101/shreya-design-cv.git
   ```
2. Navigate to the project directory
   ```sh
   cd shreya-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
